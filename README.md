        <h3 style="color: #282d33;"> 📢 한국 특허정보원 공공기관 리뉴얼 프로젝트 </h3>
        <h3 style="color: #282d33;"> 📢 제작기간 : 6일(수정 기간 포함) </h3>
        <h3 style="color: #282d33;"> 📢 리뉴얼 목적 </h3>
        <ul>
            <li>다양한 사람들이 이용하는 공공기관에서 쉽게 정보를 찾을 수 있도록 리뉴얼 하고자 함.</li>
            <li>중요한 특허 정보와 연관된 업무를 하는 사이트라는 느낌이 메인페이지에 잘 느껴지는 사이트로 만들고자 함.</li>
            <li>직관적으로사이트 이용이 가능하며, 원하는 정보를 쉽게 파악할 수 있는 사이트를 만들고자 함</li>
        </ul>
        <h3 style="color: #282d33;"> 📢 리뉴얼 방향성 </h3>
        <ul>
            <li>정보</li>
            <li>차분함</li>
            <li>사이트 구조 간결화</li>
        </ul>
